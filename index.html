Core Capabilities
Live Recognition
Real-time webcam feed with histogram-based face matching against the full member database. Instant MATCH / DENY decisions with confidence scoring.
Member Database
Full CRUD management for premium lounge members. Enroll face data via webcam or photo upload. Tier-based access control — Platinum, Gold, Silver.
Gate Control
Supervisor manual override for gate open/lock. Full access log with timestamped entries and member thumbnails. Session statistics dashboard.
Analytics
Live FPS monitoring, match rate tracking, granted vs denied counts. Colour histogram cosine similarity engine — no server required.
Multi-Camera
Automatic device enumeration with dropdown selector. Built-in error diagnostics for permission denied, camera in use, file:// protocol, and HTTPS issues.
Secure & Local
All face data stored in localStorage — zero server uploads. Works fully offline. HTTPS-ready for webcam access in any modern browser.
How It Works
01
Register Member
Add member details — name, tier, lounge access, expiry. Capture face via webcam or upload a photo.
02
Enroll Face
Face image is stored as a biometric profile. The system extracts an RGB colour histogram for fast comparison.
03
Scan at Gate
Live camera feed captures the passenger. One tap on CAPTURE & SCAN runs matching against every enrolled face.
04
Instant Decision
Access GRANTED or DENIED with confidence score, member details and gate log updated in real time.
